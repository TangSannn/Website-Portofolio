Tentang Saya

Saya adalah seorang mahasiswa jurusan Teknik Informatika di Institut Teknologi PLN yang memiliki minat passion dalam bidang pengembangan web dan desain antarmuka pengguna. Perjalanan saya di dunia teknologi dimulai sejak SMA ketika saya pertama kali belajar membuat desain sederhana.

Saat ini saya sedang menempuh pendidikan semester 3 dengan fokus mempelajari berbagai teknologi web modern, database management, dan mobile application development. Saya percaya bahwa kombinasi antara desain yang menarik dan fungsionalitas yang solid adalah kunci untuk menciptakan pengalaman pengguna yang luar biasa.

Selain coding, saya juga tertarik dengan UI/UX design, videografi, dan teknologi terbaru. Saya selalu antusias untuk belajar hal baru dan berkolaborasi dalam proyek-proyek yang menantang.
